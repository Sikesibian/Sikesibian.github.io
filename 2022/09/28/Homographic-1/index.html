同态加密的定义

各种各样的同态加密算法

  • Paillier同态加密

    • 加密方案

      • 密钥生成
        1. 随机选取两个大素数 $ p, q $ 满足 $ gcd(pq, (p-1)(q-1))=1 $ （后面的条件限制了两个素数长度相等）
        2. 计算公钥 $ n = p\cdot q $ 和私钥 $ \lambda = lcm(p-1, q-1) $
        3. 选择 $ \mathbb{Z}_{n^2}^* $ 中的随机整数 $ g(0<g<n^2) $ 使得 $ n $ 能够整除 $ g $ 的阶，作为公钥（简化版可取 $ g = n + 1 $ ）
        4. 计算私钥 $ \mu = (\mathcal{L}(g^{\lambda}(mod\,n^2)))^{-1}(mod\,n) $ ，其中算子 $ \mathcal{L}(x) = \dfrac{x-1}n $
        5. 公钥 $ (n, g) $ ，私钥 $ (\lambda , \mu) $
      • 加密
        1. 明文 $ m(0\leqslant m <n) $
        2. 选择随机数 $ r (0<r<n, r\in \mathbb{Z}_{n^2}^*) $ ，且 $ gcd(r, n) = 1 $
        3. 加密： $ c = g^mr^n(mod\,n^2) $
      • 解密
        1. 解密： $ m = \mathcal{L}(c^{\lambda}(mod\,n^2))\cdot\mu(mod\,n) $
      • 同态性质
        1. 加法同态： $ D(E(m_1)\cdot E(m_2) (mod\,n^2)) = m_1+m_2(mod\,n) $
        2. 明文乘法： $ D(E(m_1)^{m_2}(mod\,n^2)) = m_1m_2(mod\,n) $ （具有对称性）
        3. 加法数乘1: $ D(E(m_1)\cdot g^{m_2}(mod\, n^2)) = m_1 + m_2(mod\, n) $
        4. 加法数乘2： $ D(E(m_1)\cdot (n+1)^{m_2}(mod\, n^2)) = m_1 + m_2(mod\, n) $
        5. 乘法数乘： $ D(E(m)^k (mod\, n^2)) = k\cdot m (mod\, n) $

  • GM同态加密（Goldwasser-Micali）

    • 加密方案

      • 密钥生成
        1. 随机选取两个大素数 $ p, q $ ，计算 $ n = p\cdot q $
        2. 选取模 $ n $ 二次非剩余中的随机数 $ z $
        3. 公钥 $ pk = (n, z) $ ，私钥 $ sk = (p, q) $
      • 加密
        1. 明文空间为 $ {0, 1} $
        2. 对于明文 $ x \in {0, 1} $ 加密者选取随机数 $ r \in \mathbb{Z}_{n}^* $
        3. 计算密文 $ c = r^2z^x(mod\, n) $
      • 解密
        1. 判断密文是否为模 $ n $ 的二次剩余，如果是，那么明文为 $ 0 $ ，如果不是则为 $ 1 $

  • Benaloh加密

    • 加密方案

      • 密钥生成
        1. 假定每个明文块大小在 $ [0, r) $ 范围内
        2. 选择两个大素数 $ p = r * p’+1 $ 和 $ q $ （满足 $ gcd(p’, r) = gcd(q-1, r) = 1 $ ）
        3. 计算公钥 $ n = p \cdot q $ 和私钥 $ \varphi(n) = (p - 1)(q - 1) $
        4. 选择满足 $ y^{\frac{\varphi(n)}r}\not\equiv 1(mod\, n), 0\leqslant y <n $ 的$ y $
        5. 计算 $ x = y^{\frac{\varphi(n)}r}(mod\, n) $
        6. 公钥为 $ (y, n) $ ，私钥为 $ (\varphi, x) $
      • 加密
        1. 选择数 $ 0 \leqslant u<n $
        2. 计算 $ c = y^mu^r(mod\, n) $
      • 解密
        1. 计算 $ a = c^{\frac{\varphi(n)}r}(mod\, n) $
        2. DLP求解 $ m = \log_x a $

  • Shamir密钥分享算法

    • 加密方案

      • 加密
        1. 明文 $ s\in\mathbb{Z}_p $ ，$ p $ 为大素数。在 $ GF(p) $ 中任取 $ k-1 $ 个随机数 $ a_1, a_2, \cdots, a_{k-1} $
        2. 构造多项式： $ f(x) = s + a_1x + a_2x^2 + \cdots + a_{k-1}x^{k-1}(mod\, p) $
        3. 任取 $ n $ 个不同的数分别代入多项式，得到 $ n $ 个密钥对： $ (x_1, f(x_1)), (x_2, f(x_2)), \cdots , (x_n, f(x_n)) $ 并分发
      • 解密
        在 $ GF(p) $ 上解方程组
        $ \begin{matrix} f(x_1) = s + a_1x_1 + a_2x_1^2 + \cdots + a_{k-1}x_1^{k-1}(mod\, p)\\\cdots\\f(x_n) = s + a_1x_n + a_2x_n^2 + \cdots + a_{k-1}x_n^{k-1}(mod\, p) \end{matrix}$
        1. 方式1：利用拉格朗日差值公式解
        2. 方式2：利用克莱默法则（范德蒙行列式）解

    • 漏洞分析

      显然，如果在加密不同明文的过程中使用了相同的随机数和素数，就可以在知道其中一个明文的情况下解出其他明文。而有意思的是，在不知道两个明文的情况下，如果知道两个明文有着一定的线性关系，也可以据此计算出两个明文。
      设 $ A_x = a_1x_1 + a_2x_1^2 + \cdots + a_{k-1}x_1^{k-1} $ 则可知有同余方程组：
      $ \left\{\begin{aligned}&s_1+A_x-f_1(x)\equiv 0\\&s_2+A_x-f_2(x)\equiv 0\\&as_1+b-s_2\equiv 0\end{aligned}\right. \,\,\,\,\,\,(mod\, p) $
