Crypto_Challenge_4

Created2022-12-19

成功通过了NeSE的升级赛，这下我也是NeSE的一员辣！

这次的升级赛题目十分有意思，感觉自己在解同余方程上的水平都进化了，接下来把我在本次升级赛中碰到的各种各样的情况，及其思考进行一个复盘。（赞美tl2cents学长！）

老规矩，还是先贴出原题：

然后列出数学关系

首先我们可以看到，这个题先生成了一个大数 $ n = p_1 \cdot q_1 $ ，然后据此生成了一个强素数因子的大数 $ N = (2k_1p_1 + 1)\cdot (2k_2p_2 + 1) $ ，这里的比特数也很讲究啊，分别生成的是2022bits（抓住了2022年的尾巴）和3072bits（根据CA /B论坛最新标准要求，从2021年6月1日起，强制要求使用不低于RSA 3072位的密钥来颁发新的代码签名证书），十分河里。

然后加密flag：

$ c = (m^{e_0}(mod\, N))^{e_0}(mod\, n) $

开头我在想是不是在这个加密的地方有有意思的变换，但是事实证明我想多了。我们只需要解下面这个模方程就可以了（因为题干上有提示# Easy modular equation system, try to solve it ）不过我还是想到了之前的一道强素数分解的题，此处略。

最后就是这一组模方程了：

$ (N + p) ^ e \equiv c_1 (mod\, n) $
$ (N + q) ^ e \equiv c_2 (mod\, n) $
$ (2N + p + q + 20221218) ^ e \equiv c_3 (mod\, n) $

分解 $ n $

Method 1

Frankly speaking, 拿到这道题我的第一反应就是通过一系列模变换得到 $ p $ 或者 $ q $ 的倍数，然后用gcd解决，最后证明，我想多了:( 呜呜呜。

不过，我还是变换出了下面一组有意思的关系：

$ c_1 + c_2 \equiv N^e + (N + p + q)^e $
$ c_1 c_2 \equiv N^e (N + p + q)^e $
$ c_3 \equiv (N + (N + p + q) + 20221218) ^ e $

随后我用 $ x = N $ 和 $ y = N + p + q $ 分别代入就有：

$ x ^ e + y ^ e - (c_1 + c_2) \equiv 0 $
$ x^e y^e - (c_1 c_2) \equiv 0 $
$ (x + y + 20221218) ^ e - c_3 \equiv 0 $

观察上面的方程，有个一次方程，这个一次方程带给我么的信息就恰好是 $ 2N + p + q (= x + y) $ 在模 $ n $ 意义下的值。而对于那个二次方程，我们可以知道这个二次方程的两个根应该就恰好是 $ x $ 和 $ y $ 二者的值。

然后你考察比特数发现这个方程并不好解。。。

Method 2

很好，现在换下一个思路：

这个思路有意思啊，也是我在下午突然想到的，最后做出来的思路。

我们可以发现，$ p_1 $ 和 $ k_1 $ 以及 $ q_1 $ 和 $ k_2 $ 是绑定的，我们可能可以通过一些变换把其中某一组消除掉，在模运算中我们可以利用 $ n = p_1q_1 $ 这个条件

就有：

$ c_1p^e \equiv ((2k_1+1)p^2 + 2k_2pq +p)^e \equiv ((2k_1+1)p^2 + p)^e $
$ c_2p^e \equiv (2k_1p^2 + (2k_2 + 1)pq + p)^e \equiv (2k_1p^2 + p)^e $
$ c_3p^e \equiv ((4k_1 + 1)p^2 + (4k_2 + 1)pq + 20221220p) ^ e \equiv ((4k_1 + 1)p^2 + 20221220p) ^ e $

结果如下：

半分钟，还不错。容易证明的是对于第一个多项式，其23次项系数（设为 $ \alpha $ ）在对称性的保证下，这时不妨假设 $ p < q $ ，应该恰好满足 $ q | (p + \alpha) $ 以及 $ p | (q - \alpha) $ 。 于是就有 $ n | (p^2 + \alpha p) $ 和 $ n | (q ^ 2 - \alpha q) $ ，即 $ n | (p + q)(p - q + \alpha) $ 。因此 $ q - p = \alpha $

然后构造一元二次方程解方程就可以把 $ p $ 和 $ q $ 解出来了。

解 $ k_1 $ 和 $ k_2 $

Method 1

续上面的 Method 3

这个方法还有一个好处就是，我们可以顺便把 $ k_1 $ 和 $ k_2 $ 解出来。 观察我们得到的多项式：

$ q^2 + aq \equiv 0 $

$ (k_1 + b_1)q + b_2k_1 + b_3 \equiv 0 $

$ (k_2 + c)q \equiv 0 $

$ k_1k_2 + d_1k_1 + d_2k_2 + d_3 \equiv 0 $

由于 $ k_1 $ 和 $ k_2 $ 的比特位数都比 $ q $ 要小，所以我们选取上面的第二个式子，让它们去模掉 $ q $ ，在模 $ q $ 意义下解方程就可以求出 $ k_1 $ ，而对于上面的第三个式子，我们可以两边同时除以 $ q $ ，这样就可以将 $ k_2 + c $ 放到模 $ p $ 意义下，从而 $ k_2 $ 也可以求出来了。

最后直接解密即可

Method 2

其实我们解出 $ p $ 和 $ q $ 之后就可以直接把 $ m_1 $ 和 $ m_2 $ 解出来，然后就可以得到： $ 2k_1p + 2k_2q $ 的值。我们把这个式子分别映到GF(p)和GF(q)上就可以了。
